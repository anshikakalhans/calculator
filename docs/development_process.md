# Development Process

1. Started by creating a new folder and opening it in VS Code.
2. Wrote the calculator program using Python.
3. Tested each operation for correct output.
4. Initialized a Git repository using `git init`.
5. Created multiple commits with meaningful messages.
6. Added .gitignore to exclude unnecessary folders.
7. Created README.md with project details.
8. Created documentation files inside a `docs` folder.
9. Will push the project to GitHub after documentation is complete.
